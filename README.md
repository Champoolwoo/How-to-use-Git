# How-to-use-Git
###1.Git Add
```bash
>git add .
```
###2.commit
```bash
>git commit -m "ข้อความที่ต้องการ"
```

###3.ถ้ายังไม่ได้ทำการเชื่อมต่อ ใช้คำสั่งนี้
```bash
>git config --global user.email "you@example.com"
>git config --global user.name "Your Name"
```

###4.การ push
```bash
>git push origin master
```
